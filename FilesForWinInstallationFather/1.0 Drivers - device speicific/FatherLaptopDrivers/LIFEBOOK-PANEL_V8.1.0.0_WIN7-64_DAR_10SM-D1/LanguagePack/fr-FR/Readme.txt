===================================================================
 Présentation du tableau d'applications LifeBook
===================================================================

Le tableau d'applications est une fonction pratique de votre ordi-
nateur portable LifeBook qui vous permet de lancer des applications
simplement en appuyant sur un bouton.

===================================================================
 Installation du tableau d'applications LifeBook
===================================================================

 - Pour installer le tableau d'applications LifeBook, vous devez 
   ouvrir une session Etitre d'administrateur (mode Administrateur).

 - Pour installer le tableau d'applications LifeBook, exécutez le 
   programme Setup.exe, puis suivez les instructions apparaissant dans
   la fenêtre de configuration du tableau d'applications LifeBook.

 - Si vous désirez réinstaller le tableau d'applications LifeBook, 
   vous devez d'abord désinstaller la version actuelle du tableau 
   d'applications LifeBook. 

===================================================================
 Désinstallation du tableau d'applications LifeBook 
 (suppression du programme installE
===================================================================

 - Pour désinstaller le tableau d'applications LifeBook, vous devez
   ouvrir une session Etitre d'administrateur (mode Administrateur). 
   En outre, pour désinstaller cet utilitaire, aucun autre utilisateur
   ne doit être connectEau système.

 - Avant de commencer la désinstallation, fermez toutes les appli-
   cations ouvertes.

 - Pour désinstaller l'utilitaire, exécutez les opérations 
   suivantes :

  a) Cliquez sur Démarrer -> Paramètres -> Panneau de 
      configuration.
  b) Dans le Panneau de configuration, sous Programmes, cliquez sur
      « Désinstaller un programme ».
  c) Dans la liste affichée, choisissez « Tableau d'applications 
      LifeBook ». Cliquez sur le bouton [Modifier ou supprimer] 
      (ou [Désinstaller] si vous utilisez Vista). Vous voyez alors 
      apparaû‘re une fenêtre « Configuration du tableau d'applications 
      LifeBook ».
  d) Fermez le Panneau de configuration, puis cliquez sur 
      [Suivant]. La désinstallation débute.

===================================================================
 Mises en garde
===================================================================

 - Dans l'assistant de configuration du tableau d'applications 
   LifeBook, il est possible que les éléments affichés après avoir 
   choisi « Sélectionner dans le menu Démarrer » soient différents de 
   ceux figurant dans votre menu Démarrer.

 - Après l'installation, les touches de raccourci « Touche Windows 
   + 3 » et « Touche Windows + 4 » sont reconfigurées avec les para-
   mètres du tableau d'applications LifeBook.
