Manuel de l’utilitaire économiseur d’énergie

Avec l'utilitaire Économiseur d’énergie, vous pouvez aisément mettre votre PC en mode d’économie d’énergie. Ce mode permet de réduire la consommation d’énergie et ainsi d'étendre la longévité de la batterie

Ce que permet l’utilitaire Économiseur d’énergie

  • Vous pouvez passer du mode d’économie d’énergie au mode normal et vice-versa en utilisant le bouton Économie d'énergie.Aller aux instructions
  • Pour ce faire, cliquez sur l'icône correspondante située dans la zone de notification, dans le coin inférieur droit de l'écran.Aller aux instructions
  • Vous pouvez paramétrer les régimes de fonctionnement pour chacun des périphériques en mode d'économie d'énergie.Aller aux instructions

Paramétrage des régimes de fonctionnement pour chacun des périphériques en mode d'économie d'énergie

Cette configuration vous permet de régler le niveau de luminosité de l'écran ou de consommation d'énergie pour chacun des périphériques en mode d'économie d'énergie. Elle vous permet également d'afficher ou non un message à l'écran lorsque vous passez d'un mode à l'autre ou lorsque vous démarrez l'ordinateur à partir du mode d'économie d'énergie. Si le périphérique ne fonctionne pas correctement, vous pouvez l'initialiser sous ce mode.
  • Seul un utilisateur possédant des droits d'administrateur peut configurer le mode d'économie d'énergie. Cela n'est pas possible pour un utilisateur aux droits limités.

  1. Double-cliquez sur l'icône Utilitaire Économiseur d'énergie(/) située dans la zone de notification, dans le coin inférieur droit de l'écran. La fenêtre [Configuration de l'utilitaire Économiseur d'énergie] apparaîtra.
  1. Paramétrez les régimes de fonctionnement pour chacun des périphériques en mode d'économie d'énergie.
    1. Cliquez sur la case [Régime en mode d'économie d'énergie] du périphérique à paramétrer.


apparaît alors sur le côté droit de la case sélectionnée.

    1. Cliquez sur , puis sélectionnez un régime en mode d'économie d'énergie.

  1. Précisez si vous souhaitez voir s'afficher un message à l'écran lorsque vous passez d'un mode à l'autre ou lorsque vous démarrez l'ordinateur à partir du mode d'économie d'énergie.

    1. "Afficher un message lorsque le système démarre en mode d'économie d'énergie"

Si vous cliquez sur et l'incorporez dans , les paramètres de chaque périphérique en mode d'économie d'énergie s'afficheront lorsque Windows Vista démarrera à partir de ce mode.
 

    1. "Afficher un message de confirmation lors du passage d'un mode à un autre."

Si vous cliquez sur et l'incorporez dans , les paramètres (fonctionnalités réduites) de chaque périphérique en mode d'économie d'énergie s'afficheront lors du passage du mode normal au mode d'économie d'énergie ; parallèlement, les paramètres (restriction des fonctionnalités annulée) de chaque périphérique en mode normal (état libre) s'afficheront lors du passage du mode d'économie d'énergie au mode normal.

    1. "Afficher un message lorsque le changement de mode a échoué."

Passage entre mode d'économie d'énergie et mode normal

Vous pouvez passer du mode d'économie d'énergie au mode normal et vice-versa de deux façons :

  • En appuyant sur le bouton Économie d'énergie sur l'ordinateur.Aller aux instructions
  • En cliquant sur l'icône correspondante dans la zone de notification, située dans le coin inférieur droit de l'écran.Aller aux instructions

  • Seul un utilisateur possédant des droits d'administrateur peut passer d'un mode à l'autre. Un utilisateur aux droits limités ne le pourra pas.
  • Si un périphérique est en cours d'utilisation, il faut d'abord l'arrêter avant de passer d'un mode à l'autre. Dans le cas contraire, il peut être impossible d'arrêter ce périphérique.

En cas d'échec du changement de mode

Si vous avez paramétré le système pour qu'il affiche un message lors du changement de mode conformément à "Paramétrage du régime de fonctionnement pour chacun des périphériques en mode d'économie d'énergie", le message de confirmation suivant apparaîtra. Lorsque le message s'affiche, vous pouvez vérifier les informations sur le périphérique qui pose problème.
Après avoir confirmé le contenu, cliquez sur [OK].
  • Si vous cliquez sur pour « Afficher un message lorsque le changement de mode a échoué. » et l'incorporez dans , cette fenêtre n'apparaîtra plus par la suite bien que le changement de mode ait échoué.

Si le changement de mode est opéré alors qu'un périphérique (ex. lecteur mixte S, lecteur CD/DVD) est en cours d'utilisation, le message suivant risque d'apparaître :

Le cas échéant, cliquez sur le bouton [OK].

Si le PC démarre en mode d'économie d'énergie

Si vous arrêtez Windows Vista en mode d'économie d'énergie, le système redémarrera dans ce même mode et une fenêtre affichant les fonctionnalités limitées pour chaque périphérique apparaîtra.

Après avoir confirmé le contenu, cliquez sur [OK].
  • Si vous cliquez sur pour « Afficher un message de confirmation lorsque le système démarre en mode d'économie d'énergie. » et l'incorporez dans , cette fenêtre n'apparaîtra plus par la suite, bien que vous démarriez Windows Vista en mode d'économie d'énergie.

Description

Le PC est en mode normal.

Mises en garde

  • Bien qu'il soit possible d'utiliser un PC avec des comptes d'utilisateur multiples, le mode d'économie d'énergie offre une seule configuration possible par ordinateur. Ainsi, si un autre utilisateur ouvre une session sur l'ordinateur avec un compte différent en mode d'économie d'énergie, le système continue dans le même mode.
  • Le passage d'un mode à l'autre ne peut être effectué que selon les instructions fournies dans le présent manuel. Le fait de mettre le PC en veille/reprise/veille prolongée ou en réinitialisation/déconnexion ne changera jamais automatiquement le mode de consommation d'énergie actuel.
  • Si la condition matérielle est modifiée intentionnellement pendant ou après le changement de mode, le logiciel ne réinitialise pas les paramètres matériels.
