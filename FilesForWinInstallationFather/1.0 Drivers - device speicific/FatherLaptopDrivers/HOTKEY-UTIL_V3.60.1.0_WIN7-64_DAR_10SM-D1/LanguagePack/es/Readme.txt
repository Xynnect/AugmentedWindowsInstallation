===================================================================
 Introducción a la Herramienta de Teclas de Acceso Rápido
===================================================================

 La Herramienta de Teclas de Acceso Rápido proporciona indicadores 
 en pantalla de las teclas de acceso rápido (Fn+F3, F4, F6, F7, 
 F8, F9*).

 * Las funciones de las teclas de acceso rápido correspondientes 
   dependen del modelo o de la configuración del BIOS de la 
   computadora.

===================================================================
 Instalación de la Herramienta de Teclas de Acceso Rápido
===================================================================

 1. Para instalar la Herramienta de Teclas de Acceso Rápido, debe 
    iniciar sesión como Administrador (Modo de Administrador) con 
    una cuenta de Administrador.

 2. Antes de usar la Herramienta de Teclas de Acceso Rápido, instale 
    el controlador de dispositivo FUJ02B1 por separado. Para obtener 
    instrucciones sobre la instalación del controlador de dispositivo 
    FUJ02B1 (versión 1.23 o posterior), consulte las Instrucciones 
    del software FUJ02B1.

 3. Antes de instalar la Herramienta de Teclas de Acceso Rápido, 
    cierre todas las aplicaciones abiertas.

 4. Para instalar la Herramienta de Teclas de Acceso Rápido, ejecute 
    el archivo Setup.exe y siga las instrucciones que aparecen en 
    la ventana de configuración de la Herramienta de Teclas de 
    Acceso Rápido. 

 5. Para reinstalar la Herramienta de Teclas de Acceso Rápido, 
    primero debe desinstalar la Herramienta de Teclas de Acceso 
    Rápido actualmente instalada. 

===================================================================
 Desinstalación de la Herramienta de Teclas de Acceso Rápido 
 (Cómo eliminar el software instalado)
===================================================================

 1. Para desinstalar la Herramienta de Teclas de Acceso Rápido, debe 
    iniciar sesión como Administrador (Modo de Administrador) con 
    una cuenta de Administrador. También debe cerrar las sesiones 
    de todos los usuarios antes de desinstalar la Herramienta.

 2. Antes de proceder con la desinstalación, cierre todas las 
    aplicaciones abiertas.

 3. Para desinstalar la Herramienta, siga estos pasos:
    a) Vaya a Inicio -> Panel de control.
    b) Haga clic en "Desinstalar un programa" del Panel de control.
    c) En la lista que se muestra, elija "Herramienta de Teclas 
       de Acceso Rápido". Haga clic en el botón [Desinstalar], y se 
       quitará la Herramienta de Teclas de Acceso Rápido.

===================================================================
 Precauciones
===================================================================

 - Cuando se inicia Windows después de cambiar la configuración 
   de pantalla o de mostrar el protector de pantalla, no habrá 
   indicadores de teclas de acceso rápido con el uso de las teclas 
   de acceso rápido (Fn+F3,F4,F6, F7,F8,F9); no obstante, las teclas 
   de acceso rápido funcionarán.

 - Cuando se activa Windows con privilegios de Administrador, no
   aparece el indicador de volumen. Sin embargo, se pueden modificar
   el brillo de pantalla, el volumen del audio o las
   condiciones del dispositivo señalador integrado. 

 - Según la condición del sistema, el indicador puede aparecer 
   detrás de otra ventana. Si eso ocurre, reinicie el sistema.  
 
 - Aunque el indicador se muestre únicamente en 26 niveles 
   diferentes, el volumen del audio controlado por un teclado o un 
   control remoto tiene 100 niveles. Como consecuencia, es posible 
   que el indicador no muestre un cambio de nivel en respuesta 
   a cada cambio realizado mediante el teclado o el control remoto.
