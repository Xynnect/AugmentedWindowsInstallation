 Información general
=====================================================================

 La Herramienta de Extensión del Sistema de Fujitsu es una aplicación 
 de software que admite funciones de extensión del sistema para las 
 computadoras portátiles de la serie Fujitsu Lifebook.

=====================================================================
 Instalación de la Herramienta de Extensión del Sistema de Fujitsu 
=====================================================================

 - Primero debe iniciar sesión como Administrador para instalar
   este software. 
 - Antes de instalar la Herramienta de Extensión del Sistema de Fujitsu,
   debe instalar el controlador de dispositivo FUJ02E3 (versión 1.00 
   o posterior) por separado. Para instalar el controlador de 
   dispositivo FUJ02E3, consulte el archivo Readme (Léame) de dicho 
   producto.
 - Al ejecutar el archivo Setup.exe, se instalará la Herramienta 
   de Extensión del Sistema de Fujitsu. Cuando se lo ejecuta, un cuadro 
   de diálogo con instrucciones aparece en la pantalla. Siga las 
   instrucciones del cuadro de diálogo para realizar la instalación. 

=====================================================================
 Desinstalación de la Herramienta de Extensión del Sistema de Fujitsu 
=====================================================================

 - En Windows XP, Windows Vista o Windows 7, inicie sesión como 
   Administrador para desinstalar el software. 
 - No realice la desinstalación si varios usuarios iniciaron sesión 
   en la función Cambiar de usuario de Windows XP, Windows Vista 
   o Windows 7. La desinstalación se debe hacer únicamente cuando 
   un solo usuario inició sesión.

Para la desinstalación:

 1. Antes de proceder con la desinstalación, cierre todas las 
    aplicaciones.
 2. Haga clic en Inicio -> Configuración -> Panel de Control.
 3. Haga clic en "Agregar o quitar programas" en Windows XP, o en 
    "Desinstalar un programa" en Windows Vista o Windows 7 
    (en Programas).
 4. Seleccione la Herramienta de Extensión del Sistema de Fujitsu 
    de la lista y haga clic en [Cambiar/Quitar] en Windows XP, 
    o en [Desinstalar] en Windows Vista o Windows 7. Aparecerá 
    un cuadro de diálogo de configuración de la Herramienta de 
    Extensión del Sistema de Fujitsu. 
 5. Cierre el Panel de control y haga clic en [Aceptar]. Se iniciará 
    la desinstalación. 

 Una vez desinstalada, la función de extensión del sistema de hardware 
 para la computadora portátil de la serie Fujitsu Lifebook no se podrá 
 usar. Para usar la función de extensión del sistema de la serie 
 Fujitsu LifeBook, deberá reinstalar la Herramienta de Extensión 
 del Sistema de Fujitsu.
